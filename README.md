# Git Card

Веб-приложение для просмотра профилей GitHub пользователей.

## 🔍 Что умеет
- Поиск пользователей с подсказками в реальном времени
- Просмотр профиля: аватар, bio, статистика
- График коммитов за год с подсчётом стрика
- Список репозиториев с языком и звёздами
- Тёмная и светлая тема

## 🛠 Стек
React · TypeScript · Tailwind CSS · Vite
